# English Vocabulary Question Pool

This repository contains a collection of English vocabulary questions originally sourced from the [Online Library GitBook](https://onlinelibrary.gitbook.io/english/question-pool/vocabulary).

## Content
- **7000 Vocabulary Level**: Questions designed to test upper-intermediate to advanced English proficiency.
- **Formats**: 
  - `data/questions.json`: For developers who want to build apps or tools.
  - `vocabulary_list.md`: For reading and studying directly on GitHub.

## How to Run the Quiz
You can run a simple terminal-based quiz using the Python script included.
